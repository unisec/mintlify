![FlexIP](https://imgs.unisec.com.br/unicontrol/FlexIP.png)

A FLEX IP atua como controlador central de uma pré-automação com ele é possível comandar as placas da linha FLEXHOME que são a IDC,IODC e OAC facilmente via barramento 
I2c, não apenas isso, A FLEX IP tem incluso nela os seguintes componentes como WI-FI,Ethernet,I2C Bus e o Bluetooth graças ao modulo ESP32 que os suporta nativamente, 
esses recursos podem ser ativados e usados em diferente contextos. E ah claro, não vamos esquecer que o Home Assistant possui integração nativa com os módulos FLEXHOME.

## Basic Configuration

```yaml
esphome:
  name: flexip
  platform: ESP32
  board: nodemcu-32s

ethernet:
  type: LAN8720
  mdc_pin: GPIO23
  mdio_pin: GPIO18
  clk_mode: GPIO0_IN
  phy_addr: 1

web_server:

api:
  reboot_timeout: 0s

ota:

status_led:
  pin:
    number: GPIO2
    inverted: yes

logger:
  level: DEBUG

output:
  - platform: gpio
    id: eth_phy
    pin:
      number: 17
      inverted: yes
```
